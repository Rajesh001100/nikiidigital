# Technology Stack

NiKii Digital is built using a modern, scalable stack to ensure performance, security, and ease of maintenance.

## Frontend (Web)
- **Framework**: [React.js](https://reactjs.org/) with [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Modular Utility-First CSS)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: React Hooks & Context API
- **Routing**: [React Router 6](https://reactrouter.com/)
- **Validation**: [Zod](https://zod.dev/)

## Backend (Server)
- **Runtime**: [Node.js](https://nodejs.org/)
- **Framework**: [Express.js](https://expressjs.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Validation**: [Zod](https://zod.dev/) (Request/Response validation)

## Database & Infrastructure
- **Primary Database**: [Supabase](https://supabase.com/) (PostgreSQL)
- **Storage**: Supabase Storage (for Materials and Syllabus)
- **Hosting**: [Render.com](https://render.com/) (Static site for frontend, Web service for backend)

## Third-Party Integrations
- **WhatsApp API**: [UltraMsg](https://ultramsg.com/) (For automated notifications)
- **Email API**: [Resend](https://resend.com/) (For official admission letters)
- **CDN**: [Cloudflare](https://www.cloudflare.com/) (For DNS and speed optimization)
